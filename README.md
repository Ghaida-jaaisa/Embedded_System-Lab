# 📘 Embedded System Lab

## 🔌 PIC Microcontroller

---

### 🔬 LAB 1  |  [📂 Open Folder](./PIC/LAB_1/)
#### 🧪 TASK #1  |  [📄 Open File](./PIC/LAB_1/TASK%20%231)
🔹 Connect the LEDs to the PIC programmer.  
🔹 Turn on the four **left** LEDs and turn off the four **right** LEDs for `0.5 seconds`.  
🔹 Then keep alternating between the right and left LEDs.

---

### 💡 LAB 2 – LEDs & Switches  |  [📂 Open Folder](./PIC/LAB_2/)
#### 🧪 TASK #1  |  [📄 Open File](./PIC/LAB_2/TASK%20%231)
🔹 Connect the LEDs to `PORTD`.  
🔹 Turn on one LED and shift it left every `400 ms` using the `<<` operation.

#### 🧪 TASK #2  |  [📄 Open File](./PIC/LAB_2/TASK%20%232)
🔹 Connect LEDs to `PORTD` and switches to `PORTC`.  
🔹 Blink all LEDs every `0.3 second`.  
🔹 Pressing button `RC0` ➜ ⏫ increases delay by `100 ms`.  
🔹 Pressing button `RC1` ➜ ⏬ decreases delay by `100 ms`.

### 🔢 LAB 3 – 7 Segment Display  |  [📂 Open Folder](./PIC/LAB_3/)
#### 🧪 TASK #0  |  [📄 Open File](./PIC/LAB_3/TASK%20%230)
🔹 Display number **59** for `3 seconds` then turn it off.

#### 🧪 TASK #1  |  [📄 Open File](./PIC/LAB_3/TASK%20%231)
🔹 Display number **44** for `3 seconds` then turn it off.

#### 🧪 TASK #2  |  [📄 Open File](./PIC/LAB_3/TASK%20%232)
🔹 Display numbers: `00, 10, 20, ..., 90` then repeat.  
🔹 Update every `0.6 seconds`.

#### 🧪 TASK #3  |  [📄 Open File](./PIC/LAB_3/TASK%20%233)
🔹 Display number **4**.  
🔹 Each press on `SW6` ➜ 🔼 Increments number (up to 9).  
🔹 Each press on `SW7` ➜ 🔽 Decrements number (down to 0).

### 🧭 LAB 4 – Report #4 (Keypad & Interrupts)  |  [📂 Open Folder](./PIC/LAB_4/)
#### 🧪 TASK #1  |  [📄 Open File](./PIC/LAB_4/TASK%20%231)
🔹 Use the keypad:  
  `Key 2` or `Key 5` ➜ Toggle **all LEDs on/off**.

#### 🧪 TASK #2  |  [📄 Open File](./PIC/LAB_4/TASK%20%232)
🔹 Keypad behavior:  
  `Key 5` ➜ Turn ON **LED 5**  
  `Key 6` ➜ Turn ON **LED 6**  
  `*` button ➜ Turn ON **all LEDs**

#### 🧪 TASK #3  |  [📄 Open File](./PIC/LAB_4/TASK%20%233)
🔹 Blink all LEDs every `0.8 second`.  
🔹 Interrupt-based control:  
  `Push button 0` ➜ ⏫ Increase blinking time by `100 ms`  
  `Push button 2` ➜ ⏬ Decrease blinking time by `100 ms`
